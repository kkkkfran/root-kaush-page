# Pagina negra minimal

Edita `script.js` y cambia `name: "TU NOMBRE"` por tu nombre real.

Para mostrar tu avatar real de Discord, cambia `discordAvatarUrl` por un link directo a tu avatar o coloca tu `discordUserId`.

La animacion escribe el nombre, espera, lo borra hacia atras y vuelve a empezar.
